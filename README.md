# Hospital-Emergency-Room-Dashboard-Excel

**Excel dashboard for analyzing ER patient flow, wait times, and operational KPIs.**

 • Built a dynamic Excel dashboard to monitor emergency room performance, patient flow, and service efficiency.
 • Analyzed trends in admissions, treatment time, and discharge rates using Pivot Tables, charts, and conditional formatting.
 • Delivered actionable insights to support hospital management in optimizing ER resource utilization
 • Utilized Pivot Tables, charts, slicers, and advanced formulas to track trends and deliver actionable insights for hospital operations.
